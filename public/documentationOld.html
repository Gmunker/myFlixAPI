myFlix API

The server-side component that will provide information about different movies to users. Users will also be able to sign-up, update their information and create a list of favorites.

Endpoint routes

  • /users - user registration - PUT/POST/DELETE
  • /users/:name - user specific profile - PUT/POST/DELETE
  • /movies - displays a list of movies - GET { "title": "The Shawshank Redemption", "rank": "1", "id": "tt0111161" }
  • /movies/:name - displays movie informtion by name - GET
  • /genres - displays all genres - GET
  • /genres/:name - displays genres by name - GET
  • /directors - displays all directors - GET
  • /directors/:name - displays a director by name - GET
